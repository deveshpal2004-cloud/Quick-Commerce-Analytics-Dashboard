# Quick Commerce Analytics Dashboard

## Project Overview
Analyzed 100,000+ orders from Blinkit, Swiggy Instamart, and JioMart using Power BI and Python.

## Tools Used
- Power BI
- Python
- Pandas

## Key Metrics
- Total Orders: 100,000
- Total Revenue: ₹59.1 Million
- Average Delivery Time: 29.54 Minutes
- Average Customer Rating: 3.24

## Key Insights
- Swiggy Instamart generated the highest revenue.
- Blinkit achieved the fastest average delivery time.
- JioMart received the highest customer rating.
- Personal Care was the highest revenue-generating category.

## Dashboard Features
- KPI Cards
- Revenue Analysis
- Category Analysis
- Delivery Performance
- Customer Rating Analysis
- Refund Analysis
- Interactive Filters
